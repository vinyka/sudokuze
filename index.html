Como Jogar SUDOKU DO ZÉ

Objetivo

Preencha a grade 9x9 com números de 1 a 9 de forma que cada linha, coluna e caixa 3x3 contenha todos os dígitos de 1 a 9 sem repetição.

Controles

  • Clique em uma célula vazia para selecioná-la
  • Use os botões numéricos ou o teclado para inserir números
  • Pressione Delete ou clique em Limpar para remover um número
  • Use as setas do teclado para navegar entre as células
  • Pressione Ctrl+N para iniciar um novo jogo

Dicas

  • Números em preto são fixos e não podem ser alterados
  • Números em azul são suas inserções
  • Números em vermelho indicam conflitos
  • Comece preenchendo os números óbvios primeiro
  • Use o botão "Novo Jogo" para reiniciar a qualquer momento

Níveis de Dificuldade

  • Fácil: Mais números preenchidos, ideal para iniciantes
  • Médio: Desafio equilibrado para jogadores intermediários
  • Difícil: Para especialistas em Sudoku
